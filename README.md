# IntroductionDeepLearningDQN

Quick guide to create a virtual environment, install dependencies, and run the sample.

## Setup
```bash
cd IntroductionDeepLearningDQN          # enter project root
python -m venv .venv                    # create virtual environment
source .venv/bin/activate               # activate (PowerShell: .venv\Scripts\Activate.ps1)
pip install .                           # install package (dev mode: pip install -e .)
```
